# Venarus

Тестовое задание на верстку макета из figma.

В макете реализовано:
* анимация волны, 
* бегущая строка, 
* микро-анимация иконок, 
* модальное окно при клике на изображение в разделе "О производстве", 
* автоматический слайдер.
